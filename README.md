## Prerequisites
+ [Oracle VirtualBox](https://www.virtualbox.org/wiki/Downloads)
+ [Vagrant](https://www.vagrantup.com/downloads.html)

## Installation
*Windows Users should perform the following steps using the Git Shell*

```shell
cd sunday
touch .env
vagrant up
```

## Execution
```shell
cd sunday
vagrant up
start http://localhost:8081/
```
